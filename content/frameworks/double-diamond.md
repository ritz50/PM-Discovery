# Double Diamond

Design Council model for discovery and delivery.

## Phases

1. **Discover** — Diverge: explore the problem space
2. **Define** — Converge: frame the right problem
3. **Develop** — Diverge: explore solutions
4. **Deliver** — Converge: ship the right solution

## PM role

- **Discover/Define** = Problem discovery (interviews, synthesis)
- **Develop/Deliver** = Solution discovery (prototypes, experiments)

## Key insight

Do not skip the first diamond. Building before defining the problem wastes cycles.
