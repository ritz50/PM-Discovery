# JTBD Four Forces of Progress

## Diagram

```
        PUSH (pain)          PULL (attraction)
              ↘               ↙
               →  SWITCH  ←
              ↗               ↖
      HABIT (inertia)    ANXIETY (fear)
```

## Switch equation

**Push + Pull > Anxiety + Habit** → customer switches

## Questions per force

**Push:** What's frustrating about today?
**Pull:** What's appealing about the new option?
**Anxiety:** What could go wrong if they switch?
**Habit:** What's comfortable about staying put?

## PM actions

- Amplify push and pull in messaging
- Reduce anxiety with trials, guarantees, social proof
- Break habits with migration tools and onboarding
