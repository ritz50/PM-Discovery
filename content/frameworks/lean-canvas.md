# Lean Canvas

One-page business model for testing assumptions.

## Blocks

1. Problem (top 3)
2. Customer Segments
3. Unique Value Proposition
4. Solution (top 3 features)
5. Channels
6. Revenue Streams
7. Cost Structure
8. Key Metrics
9. Unfair Advantage

## Discovery use

Use Problem + Segments + Solution blocks for assumption mapping. Validate riskiest blocks first with customer interviews.
