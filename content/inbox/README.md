# Content inbox

See **[INDEX.md](INDEX.md)** for how your helloPM session folders map to modules.

Drop all raw materials here before processing:

- helloPM video transcripts (`.txt`, `.md`)
- PDFs and docs → `resources/`
- Screenshots and diagrams → `resources/images/`

After Cursor processes a file into `content/modules/`, move the original to `content/archive/`.
