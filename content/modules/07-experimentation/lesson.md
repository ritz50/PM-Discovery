# Experimentation & MVP

Design the smallest test that validates your riskiest assumption.

## Hypothesis format

> We believe [segment] will [behavior] because [reason]. We'll know when [metric] reaches [threshold].

## MVP types

- **Concierge** — Manual delivery behind the scenes
- **Wizard of Oz** — Looks automated, human-powered
- **Landing page** — Measure sign-up intent
- **Feature stub** — Minimal coded slice

## Experiment design

1. State hypothesis
2. Define success metric and threshold
3. Choose MVP type
4. Set time box (1-2 weeks typical)
5. Run and decide: persevere, pivot, or kill

## Key takeaway

An experiment without a pre-committed decision rule is just a demo.
